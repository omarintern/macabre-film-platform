**User Model Requirements** [Source: docs/prd/6-epic-details.md#story-1.2]:

- **Password**: Hashed password field (security requirement from Story 1.1 context)
- **Role**: Enum field with values 'CLIENT', 'CREATOR', 'ADMIN'
- **Default Role**: New users default to 'CLIENT' role
- **Timestamps**: Created and updated timestamp tracking

**Current Schema Status**: User model already exists in `apps/api/prisma/schema.prisma` with all required fields:
```prisma
model User {
  id        String   @id @default(cuid())
  email     String   @unique
  password  String
  role      UserRole @default(CLIENT)
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
  @@map("users")
}

enum UserRole {
  CLIENT
  CREATOR
  ADMIN
}
```
