# Story 2.5: Post Connection UI

## Status
Done

## Story  
**As a** Creator,
**I want** to add connection options to my work submission form,
**so that** I can create mosaics (chains of my own posts) or references (links to other creators' posts).

## Acceptance Criteria
1. The WorkSubmissionModal includes a new "Connections" section above the tags field
2. Connection type selection offers three options: "None", "Start/Continue Mosaic", "Add References"
3. For "Start/Continue Mosaic": Interface to select own previous posts and enter meta-title
4. For "Add References": Interface to search and select other creators' posts  
5. Connection selections are optional and don't interfere with existing submission flow
6. All existing form functionality (title, body, classification, tags) remains unchanged
